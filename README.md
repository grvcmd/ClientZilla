## Getting Started
1. Download [Node.js](https://nodejs.org/en/download/) 
2. We will use Node Package Manager [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) to install our dependencies
    ```
    npm install
    ```
3. Start the server
    ```
    npm run devstart
    ```
